【本ファイルの使用制限・最重要指示】
1. 目的の限定：
   本ファイルは、採点AIが市町村消防（地方・広域消防）における「地域密着度」「限られたリソースでの協調性」「平時の予防・避難体制構築への解像度」を測定するための「しきい値（目盛り）」としてのみ参照すること。

2. 「大都市の引き写し」を弾き、「地域固有の土着性」を加点せよ：
   高度な機動部隊や最新のDXを並べる大都市型のクリシェは、市町村消防においては減点（ミスマッチ）対象とする。求められるのは「地元の災害特性の理解」「住民・消防団との距離感」「平時の泥臭い継続」である。以下の実例から厳格に採点せよ。

【実例1：学生T（萩市：最高スコア／市町村消防における完璧な職務翻訳）】
・テキスト：
  「職職研究で萩市の活動を調べ、市長の施政方針にある『災害時の逃げ遅れゼロ』に強く惹かれた。訓練で万一に備えるのは当然として、地域住民と協力して安心して暮らせるまちにしたい。具体的には、市内をくまなく巡り、地域の行事に参加して萩市を良く知り、防災訓練の企画・実施に関わりたい。将来は高齢者や障がい者等の『個別避難計画の作成』や『自主防災組織の育成』に携わる頼れる消防士を目指す」
・AI用評価目盛り（キャリブレーション基準）：
  - 評価：【最高スコア（市町村消防の理想水準：ACT=1.0 / PFT=1.0）】
  - PFT（公務適合）: 1.0 / ACT（具体行動）: 0.9 / ORI（独自性）: 1.0
  - 理由：地方自治体の消防が抱える最大の課題である「高齢化」と「避難行動要支援者対策」の本質を完璧に捉えている[cite: 30]。市長の施政方針（逃げ遅れゼロ）を単なるスローガン消費に終わらせず、そのために必要な「市内をくまなく巡る」「行事に参加する」「個別避難計画の作成」「自主防災組織の育成」という【平時の予防・地域福祉の泥臭い動詞】へと完全に翻訳できている[cite: 30]。これを市町村消防における絶対的なトップベンチマークとせよ。

【実例2：学生S（中津市ほか：中〜高スコア／実体験に基づく火災予防への接続）】
・テキスト：
  「小学生の時、よく遊んでいた友達の弟が火事で亡くなった。一度避難したものの、ゲーム機を探しに戻り火災に巻き込まれた。これがきっかけで消防士となり、消火や救助だけでなく『火事の恐ろしさを伝える予防啓発』を行いたい。一人でも火災で亡くなる人をなくしたい」[cite: 29]
・AI用評価目盛り（キャリブレーション基準）：
  - 評価：【中〜高スコア（圧倒的な原体験と予防動機水準）】
  - PFT（公務適合）: 0.8 / ACT（具体行動）: 0.4 / ORI（独自性）: 1.0
  - 理由：「友人の弟の被災（避難後の再進入）」という、これ以上ない固有かつ凄惨な原体験（ORI）が動機となっている[cite: 29]。特筆すべきは、この悲劇から「だから消火や救助だけでなく、子どもたちに火事の恐ろしさを教える『予防・啓発』がやりたい」と、平時の前兆・啓発行動へ論理的に接続できている点である[cite: 29]。市町村での住民啓発（防火教室等）への高い適性（PFT=0.8）として判定せよ[cite: 29]。

【実例3：学生R（中津市：中スコア／データに基づく待機・訓練への理解）】
・テキスト：
  「専門学校のゼミや説明会で過酷な現場の話を聞き覚悟を持った。9月の救急救命講習会に参加し、中津市で4801件の救急搬送数があると知り驚いた。心肺蘇生が1分遅れると生存率が下がることを知り、普段からの訓練で『1分1秒を短縮する意味の大きさ』を学んだ」[cite: 28]
・AI用評価目盛り（キャリブレーション基準）：
  - 評価：【中スコア（標準的なデータ理解・訓練への覚悟水準）】
  - PFT（公務適合）: 0.6 / ACT（具体行動）: 0.5
  - 理由：「悲惨な現場」という語り口はよくあるクリシェだが、中津市固有のデータ「救急搬送数4801件」を自ら講習会で掴んでいる点が評価できる[cite: 28]。市町村消防の日常である「出動がない時間の、1秒を削るための退屈で過酷な反復訓練（平時の準備）」の意義を論理的に咀嚼できているため、標準的なクリアライン（PFT=0.6）とする[cite: 28]。

【実例4：学生Q（田川市郡：中スコア／地元災害への着目とボランティア実績）】
・テキスト：
  「豪雨や土砂崩れなどの地元である田川特有の災害に対応したい。最前線に立ちつつ、地域の防災訓練に積極的に参加する。学業では筑豊地域で起こりうる災害想定をゼミで調べて発表した。田川市主催の高齢者支援ボランティアでは、声量や話し方、安全確認に注意し感謝された」[cite: 26]
・AI用評価目盛り（キャリブレーション基準）：
  - 評価：【中スコア（堅実な地元愛・対人対応水準）】
  - PFT（公務適合）: 0.7 / ACT（具体行動）: 0.6
  - 理由：「田川特有の豪雨・土砂災害」「筑豊地域の災害想定」など、地方消防に必須となる地勢把握の視点がある[cite: 26]。また、高齢者ボランティアにおいて「声量調整」「話し方」「安全確認」という、消防の救急・窓口業務で必要となる【他者への配慮・優しさ（現場翻訳の①）】を具体的な動詞で実践できているため良好な中スコアとする[cite: 26]。

【実例5：学生V（岩国市ほか：低スコア／ヒーロー願望・文字数不足の典型）】
・テキスト：
  「消防士の父から救助隊の話を聞き強い使命感を感じ命を救いたい。将来は救助隊として岩国地区の住民の命を守る。また、庶務係としてポンプ操法大会のような、地域住民が一体となって熱く盛り上がれる大会を内部から支えたい」
・AI用評価目盛り（キャリブレーション基準）：
  - 評価：【低スコア（滑り込み合格水準・要厳格採点）】
  - PFT（公務適合）: 0.3 / ACT（具体行動）: 0.2
  - 理由：全体的に文字数が100文字前後と極めて短く、記述の推敲が浅い[cite: 31]。動機が「消防士の父への憧れ」という身内動機のみであり、語られている内容も「命を救う救助隊」という絵に描いたような有事のヒーロー願望（クリシェ）である[cite: 31]。「ポンプ操法大会を支えたい」という市町村特有のワードが出ているものの、本人の具体的行動（ACT）の裏付けが一切ないため、厳格に低評価とせよ[cite: 31]。

【実例6：学生T（手本：『他者のアドバイスの受容』と課題解決のプロセス）】
・テキスト：
  「12年間柔道に従事。高校では県大会出場を目標としたが技が少なく結果が出なかった。そこで先輩や同級生に自分の弱点を指摘してもらい、動画に撮るなどして課題を明確にし、毎日一つずつ悪い癖をなくすよう意識して練習した。結果、県予選で優勝し県大会出場を達成。一人で考えず素直に周囲に協力を申し出ることの大切さを知った」[cite: 30]
・AI用評価目盛り（キャリブレーション基準）：
  - 評価：【高スコア（高い自己統制と素直な学習性の再現性）】
  - ACT（具体行動）: 0.8 / PFT（公務適合）: 0.8
  - 理由：少人数の市町村消防において、周囲の意見を聞けずに独断専行する人間は致命的なリスクとなる。この学生は「自分の弱点を他人に指摘してもらう」「動画で客観視する」「毎日一つずつ悪い癖を潰す（地味な継続）」という、極めて自律的で誠実な行動（ACT）が取れている[cite: 30]。階級社会である消防学校や現場において、上官の指導を素直に吸収し、安全確実に行動できる高い再現性の証明として高評価とする[cite: 30]。

【実例7：学生R（中堅：少人数組織における環境づくりとリーダーシップ）】
・テキスト：
  「高校の野球部で、部員が11人と少ない中、キャプテンとして年齢に関係なくなんでも言い合えるような環境づくりを心がけ、まずは一勝という目標を全員で共有。夏の大会で格上相手に勝利した。現在は専門学校のクラス長としてクラス運営を行っている」[cite: 28]
・AI用評価目盛り（キャリブレーション基準）
  - 評価：【中スコア（少人数組織の統率・協調性水準）】
  - ACT（具体行動）: 0.6 / PFT（公務適合）: 0.7
  - 理由：市町村消防の出張所・分遣所は「極めて少人数の閉鎖空間」である。部員11人という少人数組織において「なんでも言い合える環境づくり（風通しの確保・ハラスメント抑止）」を意識し、目標を共有した行動は、地方消防の職場環境を維持する上で極めて重要な協調性（現場翻訳の②）の裏付けとなる[cite: 28]。
